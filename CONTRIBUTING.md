# Contributing

Pull requests are welcome, especially new `examples/` for industries not yet covered.

## How to Contribute

1. Fork the repository
2. Create a branch (`git checkout -b my-example`)
3. Add or edit files
4. Commit with a clear message (`git commit -m "feat: add dentist office example"`)
5. Push and open a pull request

## What We're Looking For

- **New industry examples** in `examples/` — follow the format of the existing ones (step-by-step prompts, design direction notes, completion checklist)
- **Skill improvements** — clearer instructions, better defaults, additional patterns
- **Bug fixes** — broken links, typos, incorrect references

## Guidelines

- Keep instructions beginner-friendly — assume zero web design experience
- Every example needs a "You're Done When" checklist
- Test your prompts with Claude before submitting
- No Figma-specific instructions — keep everything tool-agnostic

## Reporting Issues

Open an [issue](https://github.com/deserteaglemjAEC/web-design-workflow/issues) to report problems or request new skills/examples.
